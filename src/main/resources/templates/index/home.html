Welcome to Brain Flux!


BrainFlux is a big data management and analysis system developed through a collaboration between the University of Pittsburgh School of Medicine and School of Computer and Information. The goal of the system is to (1) aggregate the qEEG data described above into a large-scale data warehouse used by time-series database, and, (2) use the warehoused data for further research and development tasks. These tasks will be both medical discovery information science and database technique improvements. Because biomedical researchers may have clinical but not programming expertise, we aimed to have an intuitive front-end interface usable by virtually anyone. For more information about the project and system, please read the manual .

To experience more features of the system such as exporting and visualization, please login as an authorized user, If you don't have an account, please apply for one.
If you need any suggestion on our system, please give us a feedback.
If you need any technique support, please contact us.
